# collect([target])

Creates a Transform stream that collects all chunks and joins them in different ways. May emits a buffer, string or an array containing all collected chunks.

- `target` `<String>`
  - `'buffer'`
  - `'string'`
  - `'array'`
  - `undefined` Auto mode.
- Returns: `<Transform>`
